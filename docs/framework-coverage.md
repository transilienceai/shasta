# Framework Coverage

## Supported frameworks

- SOC 2 Trust Services Criteria
- ISO 27001:2022 Annex A

## Coverage model

The repository combines:

- automated AWS configuration checks
- framework mappings from checks to controls
- policy templates for non-automatable controls
- human review workflows such as access reviews and evidence collection

## Important caveat

Passing automated checks does not equal certification readiness by itself. Audit outcomes still depend on operational process, retained evidence, policy quality, and control operation over time.
